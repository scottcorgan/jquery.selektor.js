# Selektor

A jQuery plugin for simple, styleable select dropdowns. Less than 1kb gzipped!

![](https://raw.github.com/chrisbarrett/skeletor/master/skeletor.jpg)

## Install

Install as Bower component

```javascrxipt

```

```

```
